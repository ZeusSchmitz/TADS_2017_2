Locadora de Fita K7 - TDK90

Requisitos de Usuário
RU01 - Cadastrar Clientes (Usuário1);
RU02 - Cadastrar fitas (Usuário1);
RU03 - Cadastrar locação (Usuário1);
RU04 - Emitir relatório de locações por dia, semana e mês (Usuário1);
RU05 - Cadastrar fornecedores (Usuário1);

Requisitos de Sistema
RS01 - Cadastrar clientes. Campos necessários:
	Pessoal
	- Nome;
	- Sobrenome;
	- Data de nascimento;
	- CPF;
	- Email;
	- Telefone; 
	Endereço 
	- CEP;
	- Logradouro;
	- Número;
	- Bloco;
	- Bairro;
	- Cidade; 
	- UF;
	- Complemento;
	(Origem: RU01)
RS02 - Cadastrar fita (RU02);
RS03 - Cadastrar locações (RU03);
RS04 - Gerar relatórios gerenciais - exemplo: rel. de clientes, fitas, locações, etc... (RU04);
RS05 - Cadastrar fornecedor (RU05)

Requisitos de Interface
RI01 - Consultar endereço do cliente e fornecedor pelo CEP quando cadastra-los (RS01, RS05);

Requisitos de Domínio
RD01 - Emitir cupom fiscal para o consumidor (Cód. de Defesa do Consumidor)
